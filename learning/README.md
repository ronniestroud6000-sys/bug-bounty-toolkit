## After you've read these, go deeper

### Free online training (in order)

1. **[PortSwigger Web Security Academy](https://portswigger.net/web-security)** — the gold standard. Do Apprentice → Practitioner. Takes 50-100 hours but you come out *significantly* better at web app testing.
2. **[Hacker101](https://www.hacker101.com)** — HackerOne's free video series + CTF labs.
3. **[TryHackMe](https://tryhackme.com)** — gamified pentesting labs (free tier is generous).
4. **[HackTheBox](https://www.hackthebox.com)** — more advanced, machine-focused pentesting.

### Books

- **The Web Application Hacker's Handbook** (Stuttard & Pinto) — dated but foundational
- **Real-World Bug Hunting** (Peter Yaworski) — walks through disclosed HackerOne reports
- **The Tangled Web** (Michal Zalewski) — deep dive into browser security model
- **A Bug Hunter's Diary** (Tobias Klein) — case studies, pre-modern but excellent storytelling

### YouTube / Twitter

Follow these and watch their videos as they drop:
- **LiveOverflow** — deep technical, lots of walkthroughs
- **NahamSec** — bug bounty workflow, program reviews
- **STÖK** — aesthetic video essays on hunting methodology
- **InsiderPhD** — accessible explanations, great for learners
- **Jhaddix** — recon methodology, Bug Bounty Village talks

### Podcasts

- **Critical Thinking - Bug Bounty Podcast** (by Justin Gardner + Joel Margolis)
- **Darknet Diaries** (security stories, less technical)

### Paid training (worth it later)

- **SANS SEC542** / **SANS SEC560** — expensive but industry-standard
- **Offensive Security OSCP** — if you want formal credentials
- **INE / eLearnSecurity** — mid-priced alternative
